screen命令

创建一个新的窗口

screen -S test

列出所有窗口

screen -ls

重新回到窗口

screen -r test

清除dead会话

screen -list会显示该会话为dead状态
使用screen -wipe命令清除该会话

远程detach某个session

screen -d test

上翻命令

同时按ctrl+a+[
